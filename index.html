NodeSchool Utrecht

Open source workshops that teach web software skills.


Workshops

We host free to attend workshops, in which workshoppers are used as curriculum and mentors help attendees work through the challenges. We host NodeSchool in Utrecht at least once a month.*

* Well, we try to host >1 per month.

For who?

Basically for everyone. The level of difficulty of our curriculum ranges from beginner to advanced, so there's something for everyone to learn.

With everyone, we mean everyone. To make sure everyone is able to feel welcome and safe, we have created a code of conduct. The shortest version is: Don't be a jerk, be excellent to eachother.

The longer version of our code of conduct provides a more subtle and elaborate perspective on our point of view.

Interested in learning Node.js?

If you'd like to learn Node.js head over to https://meetup.com/NodeSchoolUtreg, join our community and stay informed when the next meetup will be!
